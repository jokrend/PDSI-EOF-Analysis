# PDSI-EOF-Analysis
- This project analyses the Palmer Drought Severity Index (PDSI) using Empirical Orthogonal Functions (EOF) to identify patterns of variability and analyze the patterns responsible for the most variance.

## Dataset
- Dai Global Palmer Drought Severity Index (PDSI)
- Store: `https://tds.gdex.ucar.edu/thredds/dodsC/files/g/d299000/pdsisc.monthly.maps.1850-2018.fawc-1.r2.5x2.5.ipe-2.nc`

## Attribution
